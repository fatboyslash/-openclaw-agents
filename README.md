# OpenClaw Agent 角色库

> **180 个即插即用的 AI 专家人设** — 每个角色都可独立安装，一次一行命令搞定。

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![OpenClaw](https://img.shields.io/badge/OpenClaw-Compatible-blue.svg)](https://openclaw.ai)

---

## 这是什么？

这是一个为 **OpenClaw** 打造的 AI 智能体角色库，包含 **180 个专业角色**，覆盖：

- 💻 软件开发（前端、后端、DevOps、安全...）
- 📢 营销运营（小红书、抖音、微信、B站...）
- 🎮 游戏开发（Unity、Unreal、Godot...）
- 💼 商业支持（销售、法务、财务、HR...）
- 🎨 设计产品（UI/UX、品牌、视觉...）
- ...共 **17 个部门**

每个角色都有：
- **SOUL.md** — 身份人设、沟通风格、关键规则
- **AGENTS.md** — 专业能力、工作流程、交付物模板
- **IDENTITY.md** — 一句话简介

---

## 快速开始

### 安装单个角色

**一行命令，安装任意角色：**

```bash
bash <(curl -sL https://raw.githubusercontent.com/fatboyslash/-openclaw-agents/main/install-agent.sh) 角色名称
```

**示例：**

```bash
# 安装前端开发者
bash <(curl -sL https://raw.githubusercontent.com/fatboyslash/-openclaw-agents/main/install-agent.sh) 前端开发者

# 安装小红书运营专家
bash <(curl -sL https://raw.githubusercontent.com/fatboyslash/-openclaw-agents/main/install-agent.sh) 小红书运营专家

# 安装 Unity 架构师
bash <(curl -sL https://raw.githubusercontent.com/fatboyslash/-openclaw-agents/main/install-agent.sh) Unity 架构师
```

### 查看所有可用角色

```bash
bash <(curl -sL https://raw.githubusercontent.com/fatboyslash/-openclaw-agents/main/install-agent.sh) list
```

### 使用已安装的角色

在 OpenClaw 中通过 `@` 提及角色即可激活：

```
@前端开发者 帮我设计一个 React 组件
@小红书运营专家 帮我写一篇种草笔记
```

```bash
openclaw gateway restart
```

### Q: 如何查看已安装的角色？

```bash
ls ~/.openclaw/agency-agents/
```

### Q: 如何卸载角色？

```bash
rm -rf ~/.openclaw/agency-agents/角色名称
```

### Q: Windows 用户如何安装？

1. 安装 [Git for Windows](https://git-scm.com/download/win)
2. 打开 Git Bash
3. 运行安装命令
